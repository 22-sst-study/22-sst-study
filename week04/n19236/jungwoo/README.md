# 청소년 상어
https://www.acmicpc.net/problem/19236

## Step
1. 물고기 이동
2. 상어가 이동할 수 있는 자리 구함
   1. 보드 범위 안에 존재하고 살아 있는 물고기가 있는 자리
3. 대상 자리로 이동하여 물고기를 잡아먹고 상어의 위치와 방향을 수정
4. 1~3 반복. 단, 3을 수행할 때마다 물고기 정보가 들어 있는 보드를 복사하여 사용한다
