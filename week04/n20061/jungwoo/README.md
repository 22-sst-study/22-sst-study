# 모노미노도미노 2
https://www.acmicpc.net/problem/20061

## Step
1. 각 색깔에 맞는 삽입 위치 구함
2. 해당 위치에 블록 삽입
3. 점수 계산 및 행 삭제
4. 튀어나온 블록 있을 경우 처리
5. 1~4를 모든 입력에 대해 반복

## 풀이
### Blue
- 예시 그림과는 다르게 blue에 블록을 쌓을 때 좌우를 바꾸어서 계산했음
